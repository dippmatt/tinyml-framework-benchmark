This directory is intended to create a clear overview of all command line arguments and parameters for different frameworks and their individual settings.

1) Edit the generate_config_<framework>.py and set all arguments to your needs.
2) Run generate_config_<framwork>.py to create a json configuration file with those properties.
3) to generate the python command to run the framework automation, set the correct config file in run generate_command.py then run generate_command.py.
